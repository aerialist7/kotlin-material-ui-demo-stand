## Useful commands

### Clean

```sh
./gradlew clean
./gradlew --stop
```

### Build

```sh
./gradlew build
```

### Run

```sh
./gradlew jsRun -t
```

### Gradle Wrapper update

```sh
./gradlew wrapper
```
